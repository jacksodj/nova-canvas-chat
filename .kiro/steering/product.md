# Product Overview

## Bedrock Image Chat - Nova Canvas Integration

Production-ready text-to-image chat interface powered by AWS Bedrock Nova Canvas with support for on-demand, fine-tuned, and provisioned throughput models.

### Core Features
- **Image Generation**: Amazon Nova Canvas, Titan Image Generator V2
- **Text Chat**: Claude 3.5 Sonnet with streaming responses
- **Fine-Tuned Models**: Easy discovery and integration of custom models
- **Provisioned Throughput**: Support for guaranteed capacity models
- **Production Ready**: Complete CloudFormation deployment with VPC isolation
- **Cost Optimized**: VPC endpoints, no NAT Gateway needed
- **Secure**: IAM roles, encrypted storage, HTTPS termination

### Target Users
- Development teams needing AI-powered image generation
- Organizations with custom fine-tuned models
- Production deployments requiring guaranteed performance
- Teams wanting simple AWS Bedrock integration

### Key Value Propositions
- **5-minute setup** for fine-tuned models using discovery script
- **Pre-configured** with popular models (Nova Canvas, Claude, Titan)
- **Production-ready** infrastructure with monitoring and security
- **Cost-effective** serverless architecture (~$95-125/month)
